Run the Arbi video pipeline. This finds a trending real-world event, generates an animated video of Arbi re-enacting it with physical comedy, adds troll sounds and overlays, and saves the final video locally for manual upload to Shorts/Reels/TikTok.

Takes ~5 minutes, costs ~$0.77 per run.

Execute the pipeline by running:

```bash
python3 main.py
```

After the run completes, report the results: event title, final video path (in the output/ directory), and any errors. Check the logs directory for the JSON summary.
